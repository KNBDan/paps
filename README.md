# Описание проекта:
Чат бот для конференции, в котором необходимо реализовать пользовательский интерфейс для студентов, волонтеров и админов проекта. 
У студентов должна быть возможность получать всю актуальную информацию о конференции.
Волонтеры должны иметь возможность фиксировать посещения студентами различных мероприятий.
Админы должны иметь возможность управлять событиями на мероприятии, назначать волонтёров.
