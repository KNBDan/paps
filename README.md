# Сдача лабораторных работ.
Лабораторные работы выполнены соответствующих их названиям папках проекта.
## Описание проекта:
Чат бот для конференции, в котором необходимо реализовать пользовательский интерфейс для студентов, волонтеров и администраторов проекта. 
У студентов должна быть возможность получать всю актуальную информацию о конференции.
Волонтеры должны иметь возможность фиксировать посещения студентами различных мероприятий.
Админы должны иметь возможность управлять событиями на мероприятии, назначать волонтёров.
Представители компании будут иметь QR код с информацией о компании на своем стенде, заинтересованы в большой посещаемости студентами.
